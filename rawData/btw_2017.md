# Codebuch – Bundestagswahl_2017_Alter_und_Geschlecht.csv

Repräsentative Wahlstatistik · Bundestagswahl 24. September 2017

---

## 1. Allgemeine Beschreibung

Die Datei enthält Ergebnisse der **repräsentativen Wahlstatistik** zur Bundestagswahl vom 24. September 2017. Sie weist die abgegebenen Erst- und Zweitstimmen differenziert nach **Bundesland**, **Geschlecht** und **Geburtsjahresgruppe** (Altersgruppe) aus. Die Daten stammen vom Bundeswahlleiter und basieren auf einer Stichprobe repräsentativ ausgewählter Stimmbezirke – sie sind keine Vollerhebung wie die regulären Wahlergebnisse.

| Eigenschaft | Wert |
|---|---|
| Dateiname | Bundestagswahl_2017_Alter_und_Geschlecht.csv |
| Trennzeichen | Semikolon (`;`) |
| Zeichenkodierung | UTF-8 mit BOM (utf-8-sig) |
| Zeilen gesamt | 798 (inkl. Kopfzeile: 799) |
| Spalten gesamt | 16 |
| Gebietsebenen | Bund, 16 Bundesländer + Berlin-Ost / Berlin-West |
| Quelle | Bundeswahlleiter – Repräsentative Wahlstatistik BTW 2017 |
| Datenart | Hochgerechnete absolute Stimmzahlen (Stichprobe) |

## 2. Spaltenstruktur

### 2.1 Klassifikationsspalten (Spalten 1–4)

Diese vier Spalten definieren gemeinsam die Kombination, für die die Stimmzahlen in den übrigen Spalten gelten. Jede Zeile ist eindeutig durch die Kombination `Land × Erst-/Zweitstimme × Geschlecht × Geburtsjahresgruppe` bestimmt.

| Spaltenname | Typ | Beschreibung |
|---|---|---|
| `Land` | Text | Kürzel des Gebiets (Bund oder Bundesland). Siehe Abschnitt 3. |
| `Erst-/Zweitstimme` | Integer | Stimmart: `1` = Erststimme, `2` = Zweitstimme. |
| `Geschlecht` | Text | Geschlecht der Wählenden: `m` = männlich, `w` = weiblich, `Summe` = beide zusammen. |
| `Geburtsjahresgruppe` | Text | Altersgruppe nach Geburtsjahr. `Summe` = alle Altersgruppen zusammen. Siehe Abschnitt 4. |

### 2.2 Stimmspalten (Spalten 5–16)

Alle Werte sind **hochgerechnete absolute Stimmzahlen** auf Basis der repräsentativen Stichprobe. Sie sind ganzzahlig und können von den amtlichen Endergebnissen abweichen.

| Spaltenname | Beschreibung |
|---|---|
| `Summe` | Gesamtzahl der abgegebenen Stimmen (gültig + ungültig). |
| `Ungültig` | Anzahl ungültiger Stimmen. |
| `CDU` | Stimmen für die Christlich Demokratische Union Deutschlands. In Bayern (`BY`) leer, da CDU dort nicht antritt. |
| `SPD` | Stimmen für die Sozialdemokratische Partei Deutschlands. |
| `DIE LINKE` | Stimmen für DIE LINKE. |
| `GRÜNE` | Stimmen für BÜNDNIS 90/DIE GRÜNEN. |
| `CSU` | Stimmen für die Christlich-Soziale Union in Bayern e.V. Nur in Bayern (`BY`) befüllt, in allen anderen Ländern leer. |
| `FDP` | Stimmen für die Freie Demokratische Partei. |
| `AfD` | Stimmen für die Alternative für Deutschland. |
| `Sonstige` | Summe aller übrigen Parteien und Gruppierungen. |
| `dar. NPD` | Stimmen für die Nationaldemokratische Partei Deutschlands (darunter, d. h. in `Sonstige` enthalten). In einigen Ländern leer (nicht gesondert ausgewiesen). Siehe Abschnitt 5. |
| `dar. FREIE WÄHLER` | Stimmen für FREIE WÄHLER (darunter, d. h. in `Sonstige` enthalten). |

**Hinweis zu `dar.`-Spalten:** Die Spalten `dar. NPD` und `dar. FREIE WÄHLER` sind Teilmengen der Spalte `Sonstige`. Es gilt: `CDU + SPD + DIE LINKE + GRÜNE + CSU + FDP + AfD + Sonstige = Summe − Ungültig`.

## 6. Datenstruktur und Zeilenzahl

Die Datei ist im **Long Format** strukturiert: Jede Zeile entspricht einer einzigen Merkmalskombination. Die Gesamtzahl von 798 Zeilen ergibt sich aus:

```
19 Gebiete × 2 Stimmarten × 3 Geschlechtsausprägungen × 7 Altersgruppen = 798
```

Aggregierte `Summe`-Zeilen (Geschlecht = `Summe` oder Geburtsjahresgruppe = `Summe`) sind **in der Datei enthalten** und nicht aus den Einzelwerten zu berechnen, da die Werte auf einer Stichprobe basieren. Bei eigenen Berechnungen sollte Doppelzählung durch Summenzeilen vermieden werden.
