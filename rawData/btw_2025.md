# Codebuch: Bundestagswahl_2025_bereinigt.csv

**Quelle:** Die Bundeswahlleiterin, Statistisches Bundesamt (Destatis), 2025  
**Lizenz:** Datenlizenz Deutschland – Namensnennung – Version 2.0 (https://www.govdata.de/dl-de/by-2-0)  
**Datei:** `Bundestagswahl_2025_bereinigt.csv`  
**Trennzeichen:** Semikolon (`;`)  
**Encoding:** UTF-8  
**Zeilen:** 714 (ohne Header)  
**Spalten:** 16  

---

## Inhalt

Ergebnisse der **repräsentativen Wahlstatistik** zur Wahl zum 21. Deutschen Bundestag (23. Februar 2025).  
Stimmabgabe nach Geschlecht und Geburtsjahresgruppen in den Ländern – aufgeschlüsselt nach Erst- und Zweitstimme.

---

## Variablenbeschreibung

| Spaltenname | Typ | Einheit | Beschreibung |
|---|---|---|---|
| `Land` | String | – | Kürzel für Bundesland oder Aggregat (identisch mit 2021) |
| `Stimmentyp` | String | – | `Erststimme` oder `Zweitstimme` (identisch mit 2021) |
| `Geschlecht` | String | – | `m`, `w` oder `Summe` (identisch mit 2021) |
| `Geburtsjahresgruppe` | String | – | Alterskohorte (Format identisch mit 2021, Kohorten abweichend – s.u.) |
| `Summe` | Integer | Anzahl Stimmen | Gesamtzahl abgegebener Stimmen (gültig + ungültig) |
| `Ungültig` | Integer | Anzahl Stimmen | Anzahl ungültiger Stimmen |
| `CDU` | Integer (nullable) | Anzahl Stimmen | Stimmen für die CDU; NaN bei BY (strukturell) |
| `SPD` | Integer | Anzahl Stimmen | Stimmen für die SPD |
| `AfD` | Integer | Anzahl Stimmen | Stimmen für die AfD |
| `FDP` | Integer | Anzahl Stimmen | Stimmen für die FDP |
| `DIE_LINKE` | Integer | Anzahl Stimmen | Stimmen für DIE LINKE |
| `GRÜNE` | Integer | Anzahl Stimmen | Stimmen für BÜNDNIS 90/DIE GRÜNEN |
| `CSU` | Integer (nullable) | Anzahl Stimmen | Stimmen für die CSU; NaN bei allen Ländern außer BY (strukturell) |
| `Sonstige` | Integer | Anzahl Stimmen | Stimmen für sonstige Parteien |
| `dar_FREIE_WAEHLER` | Integer | Anzahl Stimmen | darunter: FREIE WÄHLER (Teilmenge von `Sonstige`) |
| `dar_BSW` | Integer (nullable) | Anzahl Stimmen | darunter: BSW (Teilmenge von `Sonstige`); NaN bei Erststimme in 10 Ländern (strukturell, s.u.) |

## Bereinigungsprotokoll

| Schritt | Beschreibung |
|---|---|
| Kommentarzeilen entfernt | Die ersten 11 Zeilen enthielten Metadaten; beim Einlesen mit `skiprows=11` entfernt. |
| BOM entfernt | UTF-8-BOM (`\ufeff`) mit `encoding="utf-8-sig"` entfernt. |
| Spaltennamen angeglichen | `Erst-/Zweitstimme` → `Stimmentyp`; `Die Linke` → `DIE_LINKE`; `dar. FREIE WÄHLER` → `dar_FREIE_WAEHLER`; `dar. BSW` → `dar_BSW`. Identisch mit 2021-Konventionen. |
| Stimmentyp codiert | 1/2 → `Erststimme`/`Zweitstimme`. |
| Geschlecht angeglichen | `m|d|o` → `m` (inhaltlich identisch mit 2021; keine Daten verändert). |
| Geburtsjahresgruppen-Format angeglichen | Bindestriche → Gedankenstrich mit Leerzeichen (` – `); `<=1955` → `1955 und früher`. Entspricht dem Format im 2021-Datensatz. |
| Spaltenreihenfolge angeglichen | Identisch mit 2021 (soweit Spalten vorhanden). |
| Datentypen korrigiert | `CDU`, `CSU`, `dar_BSW` von `float64` → `Int64`. |
| Rohdaten nicht verändert | Originaldatei unberührt. |

## Hinweise für übergreifende Analysen mit dem 2021-Datensatz

| Merkmal | 2021 | 2025 |
|---|---|---|
| Letzte Spalte (Sonstige-Aufschlüsselung) | `dar_dieBasis` | `dar_BSW` |
| Berlin | `BE`, `BE-O`, `BE-W` | Nur `BE` |
| Altersgruppen | 1997–2003, 1987–1996, … | 2001–2007, 1991–2000, … |

Die Altersgruppen überschneiden sich inhaltlich, sind aber nicht identisch – ein direkter Kohortenvergleich ist nur annäherungsweise möglich. Alle anderen Spaltennamen, Wertebereiche und Kodierungen sind vollständig kompatibel.
